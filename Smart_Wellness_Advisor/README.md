# Smart Wellness Advisor (Full Working System)

Hybrid Machine Learning + Knowledge Representation & Reasoning system.
This repository contains a full working example, ready for local run or GitHub.

## Contents
- src/: application source code
- data/: dataset CSV
- models/: trained model and encoders
- notebooks/: Colab notebook
- documentation/: PDFs and guides
- ui/: optional static templates

## Quick start (local)
```
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate  # Mac/Linux
pip install -r requirements.txt
python src/app.py
```

## Quick start (Colab)
Open `notebooks/Smart_Wellness_Advisor_Colab.ipynb` and run cells.
